# Attendance Management System

A web-based Attendance Management System developed to simplify attendance tracking for students and faculty.

## Project Overview

This project helps manage student attendance digitally instead of maintaining manual records. Faculty can mark attendance, and attendance data is stored securely in the database.

## Features

- Student attendance management
- Faculty/Admin login
- Mark attendance
- View attendance records
- Database storage using MySQL
- Easy-to-use web interface

## Tech Stack

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## Project Structure

```text
src/com        -> Java source files
WebContent     -> JSP, HTML, CSS pages
DATABASE       -> SQL database files
